URL Shortener Microservice

Example creation usage:

https://url-shortener-dw.herokuapp.com/api/www.google.com

https://url-shortener-dw.herokuapp.com/api/foo.com:80

Example creation output

{ "original_url":"google.com.tw", "short_url":"https://url-shortener-dw.herokuapp.com/1" }

Usage:

https://url-shortener-dw.herokuapp.com/1

Will redirect to:

https://www.google.com/
